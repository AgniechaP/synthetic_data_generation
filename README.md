# Synthetic data generator
## Program start
```bash
python3 main.py --coco /path/to/coco/file/annotations.json \
--library /path/to/library/images \
--input photo_name.jpg \
--output /path/to/output/photo/fil_1.png \
--position_x 10 \
--position_y 20 \
--scale 0.5
```
